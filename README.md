# Pomodoro Study Companion

An AI-powered study companion that combines Pomodoro focus sessions with automatic flashcard generation to help students study more effectively using active recall and spaced repetition.

---

## Overview

Pomodoro Study Companion is a productivity and learning tool designed to help students stay focused while studying and retain information better.

Instead of just timing study sessions, the app actively supports learning by generating flashcards from study content and reinforcing key concepts over time.

---

---

## Why This Project?

Many productivity apps focus only on time management, while many study apps focus only on content review. This project bridges both by:

- Encouraging deep focus
- Reinforcing learning with active recall
- Helping students study smarter, not longer

---

## Tech Stack (Planned)

- Frontend: Next.js
- Backend: Flask
- AI / NLP: Gemini API (for flashcard generation)

---

## Basic Workflow

1. Start a Pomodoro study session
2. Input notes or study material
3. Generate flashcards automatically
4. Focus during work intervals
5. Review flashcards during breaks
6. Track progress over time
